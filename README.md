# Thesis Figure Code

This repository contains code and figure-generation notes for my honors thesis:

Comparative Deployment of mTOR Signaling Across Cell Types During Vertebrate Regeneration

## Contents

- `Figure_6_17_Code_Reference.md`:
  Figure-by-figure code reference for thesis Figures 6-17.
  Dot Plot Images, Figures 3-5 were made in Trailmaker using data from Sousa et al. 2026 and the Igor Schneider Lab.
  Figure 2: UMAP Summary image of cell types per species is adapted from Sousa et al. 2026
  Figure 1: Conceptual Modeling of mTOR signaling during axolotl regeneration was made by the author in Google slides with images from www.biorender.com

## Notes

The repository includes code used to generate or document thesis figures across Axolotl, Polypterus, and Zebrafish datasets.

Some large source data files such as `.rds` objects are stored locally and are not included in this repository.
